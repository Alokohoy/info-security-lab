# info-security-lab

Course overview
This repository contains lab materials and notes for an Information Security course. Each lab folder documents a focused topic with short theory, commands, and practical steps.

Repository structure
- lab1–lab13: individual lab folders
- Each lab may include notes, scripts, sample inputs, and command logs

Labs (high level)
- Lab 1: Intro and basic files
- Lab 2: Shell scripting basics
- Lab 3: Toy shell in Python
- Lab 4: Basic server + web examples
- Lab 5: Task file
- Lab 6: Misc exercises
- Lab 7: Email/script tasks
- Lab 8: Vim basics
- Lab 9: Package management (APT)
- Lab 10: NGINX configuration + proxying
- Lab 11: Installing packages from source
- Lab 12: Brute-force demo (local, authorized)
- Lab 13: Keylogger demo (local, authorized)

How to use
1) Open a lab folder
2) Read notes and task files
3) Run commands carefully on a safe, authorized environment

Ethics and safety
Some labs demonstrate security techniques (brute-force, keylogging). Use them only for educational purposes and on systems you own or have explicit permission to test.

Notes
- Commands are written for Ubuntu/Debian; adjust for other systems.
- Output may vary by OS and installed versions.
